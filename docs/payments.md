# Payments

AI Shadow Writer uses Telegram Stars for access control.

## How Access Works

The bot includes a free usage layer and paid access plans. Once free usage is exhausted, the user can unlock continued access through Telegram Stars.

## Available Plans

The current plan structure is:
- 14 days
- 30 days
- 90 days

The exact pricing can be adjusted by the bot owner at runtime.

## What A Subscription Unlocks

Paid access unlocks continued bot usage after the free allowance is exhausted. In practical terms, it is the access layer for the product workflow, including generation and refinement features available to the user account.

## What This Is Not

Telegram Stars access is not the same as:
- an equity purchase
- a transferable software license
- a perpetual ownership grant

It is an access purchase for using the bot service during the selected period.

## What Happens After Payment

After successful payment:
- the bot receives the payment event
- access is extended for the selected duration
- the user can continue using the bot

The bot can also show current access status through the in-bot status flow.

## Refund Note

Do not describe bot access as generally refundable by default. Payment handling depends on Telegram Stars mechanics and product policy. If users have a payment issue, they should contact support rather than assume that every payment is automatically reversible.

## Payment Support

Use the bot's support and contact flows if there is a payment problem. For direct creator contact details, see [Contact](contact.md).
