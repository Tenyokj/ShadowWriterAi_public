## Product Summary

AI Shadow Writer helps you:
- connect your own Groq API key
- connect a Telegram channel
- store channel context and editorial memory
- generate drafts inside Telegram
- refine text and visuals before publishing
- publish directly to the connected channel

It is built as a proprietary product, with a public-facing documentation layer and a private source codebase.
